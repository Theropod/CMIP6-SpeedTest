Other Information

What data we collect

At the end of the test, the following data is collected and stored on the server:

  • Test ID (This ID is automatically assined by the test program. It will be shown in the "Result URL" after the tests are finished. You only need to click that part, and the Result URL will be automaitcally copied. You may then paste it in your email when asking for the removal of your testing result from the server.)
  • Time of testing
  • Test results (extact download speed, and results reported by ping and jitter program)
  • IP address
  • ISP information
  • Approximate location (inferred from IP address, not GPS)
  • User agent and browser locale

Data removal

If you prefer to have them completely removed from the testing records centrally maintained on the server, please send Mr. Yufu LIU an request to liuyufu18@mails.tsinghua.edu.cn. Please provide the ID of the test (you can get this test link from test result table) or the IP address of your computer.
